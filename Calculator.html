C++ Calculator Application

Simple but fully-functional calculator application built in Visual Studio using C++.

The application is able to perform all of the normal functions of a calculator.

The application was made using C++ along with CLR, CLI and the .NET Framework.

GitHub Repository: https://github.com/dwb1118/CalculatorProject

  • Languages: C++

  • Developer Tools: Visual Studio 2022

  • Frameworks: .NET Framework
