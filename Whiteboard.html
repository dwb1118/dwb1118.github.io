Whiteboard

Whiteboard was a Java application project designed and completed with a group of four other students.

It was designed to help grade school teachers manage classroom activities and work with ease.

The application allowed an instructor create a class, ... to create seating charts, record attendance, log behavior, and track grades for a classroom of students.

  • Language: Java
