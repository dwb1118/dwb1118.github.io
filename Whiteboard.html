Whiteboard: Teaching Assistant

Whiteboard is a Java application designed and completed with a group of four other students.

The application was designed to help teachers and instructors manage classroom activities and student information with ease.

Whiteboard allows users to:

  • Create and Manage Classrooms

  • Load and Save Classrooms

  • View and Manage Student Information

  • Log Attendance

  • Manage Assignments

  • Track Student Grades

  • Log Behavior

  • Create Seating Charts

GitHub Repository: https://github.com/Helpfullinx/WhiteBoardApp

  • Language: Java

  • Developer Tools: Eclipse, GitHub, Astah
